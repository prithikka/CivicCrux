# CivicCrux

CivicCrux is a civic issue reporting and resolution web app that helps citizens report local problems to the right authorities and track progress until resolution.

## Goal
Build a simple, mobile-friendly platform where:
- Citizens can report civic issues with photos and location.
- Officers can review and update complaints.
- Admins can monitor wards, complaints, and performance.

## Main Users
- Citizens
- Officers
- Admins

## Core Features
- Citizen signup/login
- Officer login
- Complaint submission
- Photo upload
- Location capture
- Complaint tracking
- Status updates
- Verification before closure
- Admin analytics
- Tamil and English support

## Project Focus
The first version should be simple, practical, and easy to demo as a college project.

## Tech Stack
- Frontend: React
- Backend: Node.js + Express
- Database: MongoDB
- Styling: CSS / Tailwind
- Authentication: JWT or OTP-based flow
