# CivicCrux UI Plan

## Pages
- Home page
- Citizen login
- Officer login
- Citizen dashboard
- Complaint form
- Complaint details page
- Officer dashboard
- Admin dashboard

## UI Style
- Simple
- Clean
- Mobile-friendly
- Easy to navigate
- Government-service style look

## Main Components
- Navbar
- Sidebar or bottom navigation
- Complaint cards
- Status timeline
- Upload section
- Language switcher
- Analytics cards

## Design Rules
- Use large buttons on mobile.
- Keep forms short.
- Show clear labels.
- Avoid clutter.
- Use readable colors and spacing.
