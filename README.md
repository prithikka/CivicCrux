# CivicCrux

CivicCrux is a MERN-based civic issue reporting application designed to help citizens report public issues such as garbage dumps, potholes, water leakage, and streetlight failures. Users can submit complaints with photos and location details, while officers manage and resolve issues through status updates and proof uploads. Admins can monitor complaint analytics and overall system activity.


## Features

- User Registration & Login
- Role-Based Access Control
  - Citizen
  - Officer
  - Admin
- Complaint Submission with Image Upload
- Location-Based Complaint Reporting
- Complaint Status Tracking
- Officer Status Updates & Proof Upload
- Admin Analytics Dashboard
- Simple and User-Friendly Interface
